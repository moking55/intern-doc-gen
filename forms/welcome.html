สวัสดีชาวโลก!

ก่อนจะเริ่มต้นใช้งานระบบกรอกฟอร์มฝึกงาน โปรดอ่านรายละเอียดดังนี้

  1. ระบบนี้ไม่ได้มีส่วนเกี่ยวข้องกับการฝึกงานของทางสาขาวิทยาการคอมพิวเตอร์แต่อย่างใด เป็นเพียงเครื่องมือช่วยกรอกข้อมูลสถานประกอบการ เท่านั้น
  2. การจะเริ่มขั้นตอนนี้ได้ต้องมีสถานที่ฝึกงานก่อน โปรดอ่านที่กลุ่ม CSMJU Internship
  3. การกรอกข้อมูลในระบบนี้จะต้องกรอกข้อมูลให้ครบถ้วน และถูกต้อง หากพบว่าข้อมูลไม่ถูกต้อง จะไม่สามารถดำเนินการต่อได้
  4. ระบบนี้จะไม่เก็บข้อมูลส่วนตัวของท่าน เพื่อความโปร่งใส สามารถตรวจสอบ source code ได้ ที่นี่

หากท่านได้อ่านและเข้าใจข้อความข้างต้นแล้ว สามารถเริ่มกรอกข้อมูลได้เลย
